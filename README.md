# Dlicom Coin Slice

Аrcade slicing mini‑game. Click “Start”, slice coins, don’t let them fall. You have 3 lives.

Play online (after GitHub Pages deploy completes):
https://01igor.github.io/dlicom-game/

Features
- Mouse slicing with trail, physics, progressive difficulty.
- HUD: Score, Lives, Time. Three lives and Game Over screen.
- Main menu with animated background and links.
- Language switcher: EN / UA / RU.
- Sounds (Web Audio, toggle in menu).
- Mobile‑ready soon: touch controls planned.

Local run
Just open index.html in a browser or serve the folder with any static server.
